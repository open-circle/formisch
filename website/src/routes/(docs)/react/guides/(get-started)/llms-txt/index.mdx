# LLMs.txt

If you are using AI to generate forms with Formisch, you can use our LLMs.txt files to help the AI better understand the library.

## What is LLMs.txt?

An [LLMs.txt](https://llmstxt.org/) file is a plain text file that provides instructions or metadata for large language models (LLMs). It often specifies how the LLMs should process or interact with content. It is similar to a robots.txt file, but is tailored for AI models.

## Available routes

We provide several LLMs.txt routes. Use the route that works best with your AI tool.

- [`llms.txt`](/llms.txt) contains a table of contents with links to all Markdown files
- [`llms-react.txt`](/llms-react.txt) contains a table of contents with links to React related files
- [`llms-react-full.txt`](/llms-react-full.txt) contains the Markdown content of the entire React docs
- [`llms-react-guides.txt`](/llms-react-guides.txt) contains the Markdown content of the React guides
- [`llms-react-api.txt`](/llms-react-api.txt) contains the Markdown content of the React API reference

> We also provide a Markdown version of every documentation page. You can access it by replacing the trailing slash (`/`) in the URL with `.md`. For example, `/react/guides/installation/` becomes `/react/guides/installation.md`.
